# Módulo: Gestión de Proyectos y Tareas (opcional)

Objetivo:
- Seguimiento de proyectos internos y tareas asignadas.

Funcionalidades MVP:
- Crear proyectos y tareas.
- Asignar responsables y fechas.
- Vista Kanban básica (opcional).

Páginas/Endpoints sugeridos:
- GET /projects
- POST /projects
- GET /projects/:id/tasks

Notas:
- Este módulo puede quedar para una segunda fase si se prefiere.

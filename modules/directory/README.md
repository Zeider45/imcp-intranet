# Módulo: Directorio de Empleados

Objetivo:
- Lista y fichas de empleados con información de contacto y orgánica.

Funcionalidades MVP:
- Búsqueda de empleados (nombre, departamento).
- Ficha pública con contacto, puesto, jefe directo.
- Integración básica con perfil de usuario.

Páginas/Endpoints sugeridos:
- GET /employees
- GET /employees/:id

Notas:
- Considerar privacidad y campos visibles según role.

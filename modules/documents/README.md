# Módulo: Documentos / Repositorio de archivos

Objetivo:
- Almacenar, compartir y controlar acceso a documentos internos.

Funcionalidades MVP:
- Subir/descargar archivos.
- Carpetas con permisos básicos.
- Búsqueda por nombre/etiqueta.
- Versionado básico (opcional en primera fase).

Páginas/Endpoints sugeridos:
- GET /documents
- POST /documents/upload
- GET /documents/:id/download

Notas:
- Definir límites de tamaño y tipos permitidos.

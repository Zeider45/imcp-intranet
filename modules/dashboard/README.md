# Módulo: Dashboard / Inicio

Objetivo:
- Proveer una vista inicial personalizada con resumen de información relevante.

Funcionalidades MVP:
- Vista con widgets: noticias recientes, tareas pendientes, próximos eventos.
- Configuración básica de widgets por usuario.
- Enlaces rápidos a módulos frecuentes.

Páginas/Endpoints sugeridos:
- GET /dashboard (datos agregados para widgets)

Notas:
- Diseñar widgets como componentes reutilizables.

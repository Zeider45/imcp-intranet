# Módulo: Mensajería / Notificaciones

Objetivo:
- Comunicación rápida entre usuarios y avisos del sistema.

Funcionalidades MVP:
- Notificaciones del sistema (in-app).
- Mensajería simple 1:1 (chat básico) o buzón interno.
- Centro de notificaciones (leer/no leído).

Páginas/Endpoints sugeridos:
- GET /notifications
- POST /messages
- GET /conversations/:id/messages

Notas:
- Priorizar notificaciones; mensajería puede ser minimal al inicio.

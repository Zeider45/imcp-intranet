# Módulo: Calendario y Eventos

Objetivo:
- Gestión de eventos corporativos y calendarios de equipo.

Funcionalidades MVP:
- Crear/editar eventos.
- Vista mensual/semanal/día.
- Invitar participantes y RSVP básico.
- Exportar evento (iCal) / integraciones futuras.

Páginas/Endpoints sugeridos:
- GET /calendar/events
- POST /calendar/events

Notas:
- Evaluar sincronización con Google/Exchange más adelante.

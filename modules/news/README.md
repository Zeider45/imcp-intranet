# Módulo: Comunicados / Noticias / Anuncios

Objetivo:
- Publicar y gestionar noticias internas y anuncios.

Funcionalidades MVP:
- Crear/editar/borrar noticias (solo roles permitidos).
- Listado de noticias por fecha y área.
- Comentarios o reacciones (opcional).
- Feed con estado leído/no leído por usuario.

Páginas/Endpoints sugeridos:
- GET /news
- POST /news
- PUT /news/:id
- DELETE /news/:id

Notas:
- Soportar imágenes y adjuntos pequeños.

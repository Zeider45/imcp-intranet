# Módulo: Formularios / Solicitudes

Objetivo:
- Formularios para solicitudes internas (vacaciones, gastos, equipos).

Funcionalidades MVP:
- Plantillas de formulario configurables.
- Envío y seguimiento de solicitudes.
- Historial y estado (pendiente, aprobado, rechazado).

Páginas/Endpoints sugeridos:
- GET /forms
- POST /forms/:id/submit
- GET /requests
